# JAVA 下的函数式编程
## 故事
众所周知，JAVA是面向对象语言，面向对象集大成者的一个语言。
面向对象对于大型项目的构建有其优势，然后有时也显得太重。
因为这些原因，JAVA引入了函数式编程的一些特性，对整个语言进行了一些增强，由此出现了project lambda，又在JAVA8中被合并进来，作为java8版本发布
注 Lambda 表达式 非正式地又可以叫 “闭包”或“匿名方法”
## 相关特性
